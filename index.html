class ModularRobot:
    def __init__(self, num_modules=9):
        self.modules = []
        self.num_modules = num_modules
        self.algorithm = "sequential"
        
    def reconfigure_to_shape(self, shape_name):
        """Reconfigures robot to specified shape"""
        targets = self.get_target_positions(shape_name)
        return self.animate_movement(targets)
        
    def get_target_positions(self, shape_name):
        if shape_name == "Line":
            return [(i, 0) for i in range(self.num_modules)]
        elif shape_name == "Square":
            side = ceil(sqrt(self.num_modules))
            positions = []
            idx = 0
            for y in range(side):
                for x in range(side):
                    if idx < self.num_modules:
                        positions.append((x, y))
                        idx += 1
            return positions

Algorithms & Methodologies

Sequential Algorithm

How it works:

Modules move one at a time in a predetermined sequence. Each module completes its movement before the next begins.

Advantages:

  • Simple to implement and debug
  • Predictable movement patterns
  • Easy collision avoidance

Disadvantages:

  • Slower reconfiguration time
  • Inefficient for large systems

Parallel Algorithm

How it works:

Multiple modules move simultaneously toward their targets. Movement occurs in synchronized phases.

Advantages:

  • Faster reconfiguration
  • Efficient resource utilization
  • Better for time-critical applications

Disadvantages:

  • Complex collision detection required
  • Higher coordination overhead

Random Algorithm

How it works:

Modules are randomly assigned to target positions and move using stochastic methods.

Advantages:

  • Good for exploration tasks
  • Simple implementation
  • No global coordination needed

Disadvantages:

  • Unpredictable completion time
  • Potential for inefficiency
  • May not find optimal paths

Movement Logic

def move_modules(modules, targets):
    for i, module in enumerate(modules):
        if i < len(targets):
            target_x, target_y = targets[i]
            # Move X then Y
            while module.x != target_x:
                module.x += 1 if module.x < target_x else -1
            while module.y != target_y:
                module.y += 1 if module.y < target_y else -1
